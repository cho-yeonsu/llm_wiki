# Gumloop

**summary**: 노코드 AI 에이전트/워크플로우 빌더 플랫폼. YouTube·Google Sheets 등 다양한 앱 연동과 프롬프트 기반 역할 지정으로 코딩 없이 자동화 에이전트를 구축할 수 있다. 유튜브 리서치 모니터링 에이전트 구축에 활용.

## 개요
Gumloop은 노코드 방식으로 AI 에이전트와 자동화 워크플로우를 만드는 플랫폼이다. 'Create Agent'로 에이전트를 생성하고, Apps 메뉴에서 외부 서비스(YouTube, Google Sheets 등)를 연동한 뒤, 자연어 프롬프트로 역할을 지정하면 동작한다. 스케줄 실행을 지원해 정기 자동화에 적합하다.

## 핵심 포인트
- **노코드 에이전트 생성**: gumloop.com/personal/agents에서 Create Agent로 즉시 생성
- **앱 연동**: YouTube(영상 검색·자막 수집), Google Sheets(결과 저장) 등 Apps 메뉴에서 활성화
- **프롬프트 기반 역할 지정**: 산업·채널·기간·출력 형식을 자연어로 지시
- **스케줄 실행 지원**: 매일 같은 시간 자동 실행 가능 → 수동 실행 대비 권장
- **대표 활용 사례**: [[유튜브_리서치_에이전트]] (영상 자막 분석 → 구조화 브리프 → 시트 저장)
