## 핵심 소비자 트렌드 5가지

### 1. "피부과 대체" 프레임이 구매 동기의 핵심
- "피부과 갈 돈이 부족하다면 디바이스 사볼만한"
- "비싼 기계가 돈값한다"
- 피부과 시술 vs 디바이스 가격 비교가 구매 결정의 핵심 트리거
- **알프랩 시사점**: 쎄라쥬가 이 프레임을 가장 잘 활용하고 있음

### 2. 단일 기기 → 복수 기기 루틴화
고관여 소비자들은 단일 기기로 만족하지 않고 기능별로 여러 기기를 조합:
```
쎄라쥬 (리프팅/하이푸) + 크라이오소닉 (홍조) + 부스터 (흡수)
```
- "기기마다 자기 영역이 있다"는 인식 확산
- 한 기기가 모든 것을 해결해준다는 기대는 줄어드는 추세

### 3. 입문 시장 양극화 — 7만원 vs 30만원
- **초저가 입문**: 페이스팩토리(7만원) "생일선물 추천"
- **중가 입문**: 메디큐브 부스터 프로(30만원) "퍼펙트 입문템"
- 두 포지션 모두 수요 있음
- 중간(10~20만원대)은 애매한 포지션

### 4. 부작용/실패 경험 공유 증가
최근 게시글에서 경고성 후기 증가:
- 풀쎄라 4.5mm → 패임 발생
- 쎄라쥬 EMS 눈밑 → 애교살 손상 위험
- "얼굴 붉어짐, 광대 따끔거림"
- **의미**: 시장 성숙기 진입, 소비자가 더 정교한 정보를 요구

### 5. 쎄라쥬는 최고 포지션이지만 진입 장벽 명확
**긍정 반응**:
- "휴대용 피부과 느낌" (조회 38,055 게시글 ⭐⭐⭐⭐⭐)
- 팔자주름, 모공, 턱선 개선 효과 실제 체감
- 카트리지 교체 불필요 → TCO 경쟁력 언급

**진입 장벽**:
- 가격 (~97만원) 
- 사용법 난이도 ("하이푸 익히기 어려움")
- 부작용 우려 (잘못 사용 시 자극)

## 가격대별 소비자 니즈 매핑

```
입문자           중간               고관여 파워유저
7~15만원         20~45만원          70~100만원+
페이스팩토리     메디큐브부스터      쎄라쥬
"그냥 한번"      "효과 보고 싶어"   "피부과 대체 확신"
흡수/갈바닉      리프팅+탄력        하이푸+고주파 복합
```

## 쎄라쥬 구매 동선 (추정)

1. 더쿠 후기글 발견 (입소문)
2. 조회수 높은 게시글 반복 확인 → 신뢰 형성
3. 가격 앵커링 확인 ("피부과 시술 vs 97만원")
4. 공식몰 방문 → 카드 무이자 확인
5. 구매 결정

## 알프랩 진입 시 시사점

- **쎄라쥬가 더쿠 소비자 마음속 1등**: 프리미엄 디바이스 시장에서 이미 강력한 포지션 선점
- **입문 시장은 아직 열려 있음**: 7~30만원대 가성비 포지션은 경쟁 가능
- **부작용 우려 해소가 차별점 기회**: "쉽고 안전한 사용법"이 아직 충족되지 않은 니즈
- **복수 기기 루틴 트렌드**: 특정 기능 특화 기기 전략도 유효
