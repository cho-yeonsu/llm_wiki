# 내 투자 자료를 '위키'로 키운다는 것

**수집일**: 2026-06-03

출처: 캬오의 일상다반사 블로그

## 핵심 메시지
흩어진 투자 자료(기사, 리포트, 메모, 캡처)를 단순히 '저장'하는 것을 넘어, LLM을 활용해 서로 연결되고 점점 풍부해지는 '위키'로 키워나가는 방법론에 대한 기록.

## 주요 내용
- 자료를 모으기만 하면 다시 찾지 못하고 죽은 데이터가 된다. 핵심은 자료 간 '연결'과 '재방문'이다.
- LLM에게 새 소스를 던져주면 기존 위키 페이지를 업데이트하고, 새로운 개념·종목은 새 페이지로 만들며, 고유명사는 위키링크로 교차연결하게 한다.
- 이렇게 쌓인 위키는 시간이 지날수록 복리처럼 가치가 누적된다 (지식의 복리).
- 기존 페이지를 덮어쓰기보다 업데이트·병기하는 방식으로 맥락을 보존한다.
- 단순 요약 소비가 아니라 직접 구조를 설계하고 검증하며 키우는 과정에서 자신만의 판단 기준(안목)이 생긴다.

## 시사점
- 위키 운영의 본질은 '검색 가능성'과 '교차참조'를 통한 지식의 복리 축적.
- AI가 생성·정리해준 내용을 그대로 받아들이지 말고, 직접 구조를 설계하고 검증·편집하는 태도가 핵심.
