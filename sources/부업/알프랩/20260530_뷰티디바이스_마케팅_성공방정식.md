## 이너시아 셀비전 — 채널별 플레이

### 채널 구성
- D2C 자사몰 (주력)
- GS샵 홈쇼핑 (확인됨)
- 공동구매 (뽀큐트몰 등 유튜버 채널 활용)
- 신라인터넷면세점 (단독 입점, 디지털리빙 1위)
- SSG, GS샵 카탈로그

### 퍼포먼스 마케팅
- 메타(Meta) 중심 광고 집행
- 네이버 검색 광고 병행
- 광고 → 자사몰 유입 → CRM으로 전환까지 퍼널 설계

### CRM (이탈 방지가 핵심)
- 웰컴 쿠폰 팝업 → 1초 카카오 로그인 → 친구추가 유도
- 장바구니 담고 3일 미구매 고객 → 카카오 친구톡 5회 이상 재독려
- "사야 하는 이유 + 지금 사지 않으면 안 되는 이유" 2가지 메시지 구성
- 카카오 친구톡 선택 이유: 문자보다 브랜드 색 표현 가능, 이메일보다 오픈율 높음

### 성과 지표
- ROAS: 10,000% 이상
- 전환율: 20% 이상
- 자사몰 매출 1년 만에 10배 성장
- 브랜드 검색량 전년 대비 5배 증가

### 가격 앵커링
- 피부과 시술 패키지 200~1,000만원 vs 쎄라쥬 97만원
- 카드 12개월 무이자 → "월 8만원대" 카피
- 홈쇼핑은 의도적으로 최소화 → 프리미엄 톤 보호

## 에이피알 메디큐브 AGE-R — 채널별 플레이

### 라이브커머스
- 부스터프로 라이브커머스 80분에 매출 50억 달성
- 미디어 커머스 역량 내재화 (광고 직접 집행)

### 채널
- D2C 자사몰 중심 + 올리브영 + 홈쇼핑(GS·CJ·현대) + 글로벌
- 데이터 중심 피드백 루프: 자사몰 → 고객 데이터 축적 → 제품/마케팅 반영

### 앵커링
- "피부과 1회 30만원 vs AGE-R 29만원"
- 카드 12개월 무이자

## 고가 뷰티 디바이스 공통 마케팅 패턴

### 채널별 역할 분담

| 채널 | 역할 | 특징 |
|------|------|------|
| **메타 DA** | 잠재 고객 인식·관심 유도 | 예산 70% 집중, 설득 단계 |
| **네이버** | 구매 확신 단계 | 예산 20%, 검색 의도 고객 |
| **배너/리타겟팅** | 이탈 유저 재상기 | 예산 10% |
| **홈쇼핑** | 단기 대규모 매출 + 신뢰도 | 단, 프리미엄 이미지 훼손 리스크 |
| **라이브커머스** | 고관여 제품 실시간 Q&A → 전환 | 80분 50억 사례 |
| **공동구매** | 인플루언서 팬덤 즉각 판매 | 수수료 30~70% |
| **면세점** | 프리미엄 이미지 + 글로벌 | 이너시아 신라면세점 1위 |
| **CRM** | 장바구니 이탈 → 전환 | 카카오 친구톡 ROAS 10,000% |

### 고가 제품 전환 공식
1. 비교 대상을 제품이 아닌 서비스로 설정 (앵커링)
2. 카드 무이자로 월 단위 체감가 낮춤
3. 권위 자산 (창업자 스토리 or 브랜드 후광)
4. 광고 유입 → CRM으로 장바구니 이탈 방어

### 채널 믹스 선택 기준
- **프리미엄 포지션 유지**: 홈쇼핑 최소화, D2C+면세점 중심 (이너시아형)
- **대중 볼륨 확보**: 홈쇼핑+올리브영 적극 활용 (에이피알형)
