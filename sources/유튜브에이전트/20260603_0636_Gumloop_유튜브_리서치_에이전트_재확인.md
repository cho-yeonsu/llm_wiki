# 10분 만에 유튜브 리서치 AI 에이전트 만들기

**수집일**: 2026-06-03

유튜브 채널을 자동 모니터링하고, 영상 내용을 분석해 요약본을 Google Sheets에 저장하는 에이전트를 Gumloop로 구축하는 방법이다.

1. 에이전트 생성
Gumloop 접속(https://www.gumloop.com/personal/agents)
Create Agent 선택
이름 지정
예: Video Brief Builder
예: Niche Monitor

2. 앱 연동
오른쪽 Apps 메뉴에서 활성화

YouTube
- 영상 검색
- 자막(Transcript) 수집

Google Sheets
- 결과 자동 저장

3. 핵심 프롬프트 입력
에이전트에게 다음 역할 부여:
"Build me a YouTube monitor for [industry]. Check [channels or search terms], find videos from the last [timeframe], read the transcript, and return a brief with title, URL, 3-5 takeaways, significance, follow-up ideas, usefulness score, and a 'what changed' summary. Log everything to a Google Sheet."

4. 분석 범위 제한
초기에는 범위를 좁게 설정
특정 분야 1개
신뢰 채널 3~5개
또는
검색어 1~2개

기간
최근 24~48시간

5. 출력 형식 고정
모든 영상에 포함:
제목
URL
핵심 내용 3~5개
중요성
후속 아이디어
유용성 점수(1~5)
무엇이 달라졌는지(What Changed)

6. 실행 및 검토
에이전트 실행 후 Google Sheets 확인

체크 항목:
링크 정상 작동 여부
핵심 내용 정확성
점수 부여 적절성
노이즈 여부

7. 점진적 확장
결과가 만족스러우면:
채널 추가
주제 확대
검색어 확대
한 번에 하나씩 추가하며 테스트

8. 팁
수동 실행보다 스케줄 실행 추천
매일 같은 시간 실행하도록 설정
범위를 넓히기 전 품질 검증 먼저 진행
